Введение

Портал электронных услуг БЖД (далее Портал электронных услуг) – онлайн сервис, обеспечивающий доступ клиентам Белорусской железной дороги к сервису «Таксировка груза». Данный сервис портала может быть использован при оценке финансовых затрат на перевозку груза и включает следующие функции:

  • определение маршрута следования и тарифного расстояния;
  • нахождение тарифа, который должен быть применен в конкретном случае;
  • исчисление провозной платы.

В перспективе Портал предусматривает возможность расширения функциональных возможностей.

Таксировка груза

Сервис «Таксировка груза» позволяет рассчитать плату за перевозку груза железнодорожным транспортом всеми видами отправок во внутреннем и международном сообщении. Данный сервис обеспечивает:

  • определение маршрута следования и тарифного расстояния;
  • нахождение тарифа, который должен быть применен в конкретном случае;
  • оперативный учет повышающих и понижающих коэффициентов;
  • исчисление провозной платы;
  • возможность расчета тарифа за указанную дату;
  • удобный поиск по коду или наименованию станции, груза;
  • детализацию расчета провозной платы;
  • формирование сводного отчета по таксировке;

Общие принципы ввода данных

Для ввода данных расчета провозной платы используется веб-страница вида, приведенного на рисунке.

При работе с визуальным интерфейсом используются следующие элементы:

Однострочное поле ввода. Для ввода текста необходимо поместить в поле курсор «мыши» и ввести текст.

Выпадающий список. Для выбора значения необходимо нажать на кнопку в правой части списка и выбрать необходимое значение.

Поле выбора из списка. Для выбора значения необходимо отметить «мышью» строку списка.

Поле для ввода даты.

Кнопки:

- кнопка открытия списка значений.

- кнопка просмотра информации по параграфам железнодорожных станций (производимые коммерческие операции).

- кнопка инверсии станций

- кнопка открытия маршрута следования.

- кнопка открытия детального отчета по таксировке груза.

- кнопка открытия сводного отчета по таксировке груза.

- кнопки выполнения расчета и очистки данных.

Все элементы ввода разделены по группам и оформлены в виде панелей, которые могут раскрываться и закрываться. Кнопка закрытия/открытия панели расположена рядом с названием панели.

Ограничения по вводу данных сервисом не предусмотрены. Ограничения по сервису Таксировка груза определяются условиями портала.

Ввод данных маршрута

На рисунке приведена панель для ввода данных маршрута.

Для ввода маршрута расчета необходимо выполнить следующие шаги.

  • Задать станцию отправления. Для этого необходимо установить фокус ввода на поле ввода станции и ввести код станции или ее наименование (можно первые символы как кода, так и наименования). При нажатии клавиши ENTER либо кнопки выпадающего списка (рядом с полем ввода) появляется панель списка. Внешний вид результата ввода станции приведен на рисунке.

    После выбора мышью в списке необходимой станции заполняется поле станции отправления, страна формируется автоматически.
  • Задать станцию прибытия. Ввод станции прибытия аналогичен вводу станции отправления. Страна также формируется автоматически.

При задании станций отправления и прибытия автоматически определяется маршрут.

При завершении расчета маршрута пользователю выдается расстояние (заполняется поле ввода Расстояние) и окно маршрута, которое доступно по нажатию кнопки с меткой Маршрут . Данная панель приведена на рисунке.

По кнопке можно просмотреть параграфы станций отправления и назначения (выполняемые коммерческие операции)

Ввод данных по грузу

Внешний вид панели для ввода данных груза приведен на рисунке.

Для ввода данных груза необходимо ввести два параметра:

  • ГНГ – значение справочника ГНГ для груза. Для ввода данного параметра необходимо установить фокус ввода в поле ввода ГНГ, ввести наименование или код ГНГ груза (можно частично) и нажать клавишу ENTER (либо кнопку выпадающего списка рядом с полем ввода). Система выдает пользователю всплывающую панель со списком релевантных значений ГНГ. Для выбора значения необходимо мышью щелкнуть левой кнопкой на позиции необходимого значения. Результатом действия является закрытие окна и заполнение поля наименования и кода ГНГ. На рисунке приведен результат ввода ГНГ. Числовой код ГНГ выставляется системой автоматически.

  • ЕТСНГ - значение для груза согласно справочнику ЕТСНГ. Ввод данного параметра аналогичен вводу значения ГНГ. После ввода значения ГНГ пользователю соответствующие коды груза ЕТСНГ.

Ввод данных по вагону

Для выполнения расчета провозной платы необходимо ввести следующие данные по вагону:

  • Тип вагона. Для ввода параметра необходимо ввести название типа (можно частично). Система выдает пользователю всплывающую панель со списком соответствующих значений. Результат ввода типа вагона приведен на рисунке 8.
  • Для выбора значения типа вагона необходимо левой кнопкой мыши нажать на необходимую позицию в списке значений типа вагона. При нажатии кнопки выпадающего cписка появляется полный список значений типа вагона.
  • Принадлежность. Для ввода параметра необходимо нажать кнопку . Система выдает пользователю панель со списком соответствующих значений. Результат приведен на рисунке. Ввод значения осуществляется путем выбора значения из списка и нажатия левой кнопки мыши.
  • Код собственника. В интерфейсе данный параметр помечен меткой КС. Для ввода данного параметра достаточно ввести в поле данных неполное название кода собственника, например «лит». В поле ввода появится значение “АО "ЛИТОВСКИЕ Ж.Д.”. Для получения полного списка значений необходимо нажать клавишу выпадающего списка.. Появляется всплывающая панель со списком значений, из которого надо выбрать необходимую позицию. Значение по умолчанию – Белорусская ж.д.

  • Оси. Данный параметр является целочисленным, заполняется через поле ввода. Значение по умолчанию – 4.
  • Проводники. Данный параметр является целочисленным, заполняется через поле ввода.

    Для завершения сеанса работы с услугой необходимо осуществить выход из портала. Для этого в заголовке портала есть пиктограмма выхода, изображенная на рисунке.
